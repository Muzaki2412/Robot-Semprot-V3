# Robot-Semprot-V3
Tim peneliti Universitas Negeri Yogyakarta (UNY) melalui hibah Program Diseminasi Teknologi Indonesia (PDTI) 2024 melakukan diseminasi inovasi teknologi pada bidang pertanian. Diseminasi ini berupa produk RoboSemp (robot semprot) versi 2, dan versi 3. RoboSemp ini dirancang untuk mengefektifkan dan mengefisiensikan penyemprotan tanaman hortikultura khususnya tanaman cabai. Implementasi teknologi robotik ini diterapkan pada Gapoktan Mugi Mulyo, Desa Sedayu, Kecamatan Muntilan, Kabupaten Magelang, Jawa Tengah dengan total 428 petani mencakup 80 ha area pertanian. Pengembangan teknologi robotik ini menjawab permasalahan yang dihadapi para petani hortikultura di daerah setempat. Tingginya biaya dan frekuensi penyemprotan tanaman cabai terlebih pada musim penghujan menjadi tantangan tersendiri bagi petani.
